CanStack Deep Dive

How stacking eligibility is determined

Stack Check Logic Pseudocode 
1. Check ID match: item1.Id == item2.Id
2. Strict Check: Name, Rarity, and Type MUST match exactly
3. Blacklist: Check Settings.RarityBlacklist (e.g. Legendary)
   - If blacklisted, return false
4. Return true if all checks pass

SpawnTool Flow

How tools are instantiated on equip

Spawn Flow Pseudocode 
1. Check player.Character exists (safety)
2. Find tool template in ServerStorage.Tools[itemId]
3. Clone template
4. Apply metadata as Attributes via SetAttribute()
5. Parent clone to player.Character
6. Return tool instance
