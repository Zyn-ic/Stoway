InventoryService

The central service managing all inventory operations

Method Parameters Description
AddItem player, itemId: string, amount: number Adds items to player inventory. Auto-stacks if possible.
EquipSlot player, slotIndex: number Equips item from hotbar slot. Spawns tool on character.
UnequipSlot NEW player Removes currently equipped tool from character.
RemoveItem NEW player, uuid: string, amount: number Removes specific amount from an item stack by UUID.
DropItem NEW player, slotType: string, slotIndex: number Drops item into world and removes from inventory.
Replicate player Syncs inventory state to client via InventoryUpdate remote.
Server Usage Example Luau 
-- Add items to inventory
InventoryService.AddItem(player, "Sword", 1)

-- Equip/Unequip
InventoryService.EquipSlot(player, 1)
InventoryService.UnequipSlot(player)

-- Remove items by UUID
InventoryService.RemoveItem(player, uuidString, 5)

-- Drop items into world
InventoryService.DropItem(player, "Hotbar", 3)

Client Remote Events

All available remotes for client-server communication

Remote Arguments Description
SwapSlots type1, slot1, type2, slot2 Swaps items between two slots
EquipSlot slotIndex Equips item from hotbar
UnequipSlot (none) Unequips current tool
RemoveItem uuid, amount Removes items from inventory
DropItem slotType, slotIndex Drops item into world
Client Usage Example Luau 
-- Swap slots between hotbar and storage
SwapSlotsRemote:FireServer("Hotbar", 1, "Storage", 5)

-- Equip/Unequip
EquipSlotRemote:FireServer(3)
UnequipSlotRemote:FireServer()

-- Drop item from slot
DropItemRemote:FireServer("Hotbar", 2)
