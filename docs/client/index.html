Hybrid Input System

Seamless switching between input methods

Client V2 introduces a robust hybrid input system managed by InteractionManager. It automatically monitors UserInputService.PreferredInput to switch modes dynamically:

  • Mouse/Touch: Enables full DragHandler functionality for traditional drag-and-drop inventory management.
  • Gamepad: Activates ConsoleModeManager, disabling drag-and-drop in favor of a state-based selection system.

Console Modes
